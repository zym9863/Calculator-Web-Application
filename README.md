# 计算器 Web 应用

这是一个简单的计算器 Web 应用，使用 HTML、CSS 和 JavaScript 实现，适合学习和演示基本的前端开发。

## 功能
- 支持基本的加、减、乘、除运算
- 响应式界面，适配不同设备
- 实时显示计算结果

## 文件结构
- `index.html`：主页面，包含计算器界面结构
- `style.css`：样式表，负责页面美化和布局
- `script.js`：实现计算器的交互逻辑

## 使用方法
1. 克隆或下载本仓库到本地。
2. 用浏览器打开 `index.html` 文件即可使用。
