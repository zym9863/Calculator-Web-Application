# Calculator Web Application

This is a simple calculator web application built with HTML, CSS, and JavaScript. It is suitable for learning and demonstrating basic front-end development.

## Features
- Supports basic addition, subtraction, multiplication, and division
- Responsive interface for various devices
- Real-time calculation results

## File Structure
- `index.html`: Main page containing the calculator UI structure
- `style.css`: Stylesheet for page layout and appearance
- `script.js`: Implements the calculator's interactive logic

## Usage
1. Clone or download this repository to your local machine.
2. Open the `index.html` file in your browser to use the calculator.
